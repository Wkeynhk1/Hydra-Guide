# Как установить общие распространяемые компоненты (Common Redistributables)

Общие распространяемые компоненты — это набор зависимостей, необходимых для корректной работы игр и установщиков. **Настоятельно рекомендуется** установить их, чтобы избежать проблем при запуске игр через Hydra Launcher.

Вот инструкция:

## Перейдите в `Настройки` > во вкладку `Совместимость`:

![Settings](assets/guides/settings.png)

![Compatibility](assets/guides/compatibility.png)

## Нажмите `Установить`:

![Install](assets/guides/install-common-redistributables.png)

::: warning Если вкладка `Совместимость` отсутствует, это означает, что у вас установлена устаревшая версия Hydra Launcher.

Вы можете скачать последнюю версию Hydra [здесь](https://github.com/hydralauncher/hydra/releases/). 
:::

## Дождитесь завершения установки.

![Installing](assets/guides/installing-common-redistributables.png)

![Installed](assets/guides/installation-completed-common-redistributables.png)

## Перезагрузите компьютер

После установки необходимо **перезагрузить компьютер**, чтобы все зависимости корректно загрузились в систему.

___

После перезагрузки основные компоненты будут успешно установлены.
