# Download failed: File not found (404) Check your connection and click 'Retry' to try downloading the files again, or click 'Cancel' to terminate setup.

![404](assets/errors/404.png)

Эта ошибка появляется, когда установщик пытается скачать файлы зависимостей, но они больше недоступны, из-за чего возникает ошибка 404.

Чтобы это исправить, запустите установку заново, но перед началом **снимите** все галочки с установки common redistributables:

![Common redistributables in FitGirl Setup](assets/errors/common-redistributables-setup.png)

После этого продолжите процесс установки.

Вы можете установить common redistributables [напрямую через Hydra](common-redistributables.md).
