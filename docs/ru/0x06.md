# Internal error 0x06: System error!

![0x06](assets/errors/0x06.png)

Эта ошибка возникает, когда в Windows отсутствуют некоторые важные папки, необходимые для работы **кряков**. Поэтому их нужно создать вручную.

Чтобы это исправить, выполните следующие шаги:

## Нажмите Windows + R, введите `%public%` и нажмите "OK":

![Windows + R](assets/errors/public-command.png)

## Создайте папку с названием `Documents`:

Щёлкните правой кнопкой мыши, выберите `Создать` > `Папку` и назовите ее `Documents`

![Create Documents folder](assets/errors/create-documents-folder.png)

![Documents folder](assets/errors/documents-folder.png)

## Откройте созданную папку и создайте внутри неё папку `Steam`:

![Create Steam folder](assets/errors/create-steam-folder.png)

![Folder Steam](assets/errors/steam-folder.png)

___

После этого запустите игру снова от имени администратора.
